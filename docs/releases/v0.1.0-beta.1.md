# v0.1.0-beta.1 - Initial Shared Lua Helper Package

`v0.1.0-beta.1` is the first published `zlua-portable` checkpoint.

This package extracts the reusable low-level Lua embedding helpers that were
previously duplicated across app repos. The goal is one shared Zig package for
generic Lua state, table iteration, and config-style table reads, while leaving
app-specific parsing policy in the consuming application.

## Included Surface

- `api.State` for Lua state lifecycle and raw stack helpers
- `api.TableIter` for generic table iteration
- `reader.Reader` for typed field reads and config-style traversal

## Intended Boundary

- generic Lua mechanics belong here
- app config semantics do not
- consumers such as `zbar` and `zide` should reuse this package instead of
  copying low-level Lua helpers locally

## Validation

- `zig build test`

## Consumption

For local sibling development, use a path dependency.

For pinned shared consumption, fetch this package by tag and save the resulting
hash in the consumer's `build.zig.zon`.
